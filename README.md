# SuRakshaPay 🛡️

**AI-Powered Financial Safety Shield for Rural India**

Built by **Harshil Parmar** & **Kush Thacker** | Hackathon 2024

---

## 🚀 Features
- 💬 SMS / Message Scam Checker
- 💳 UPI ID Verifier with inline QR Scanner
- 📷 QR Code Scanner (camera + file upload)
- 📚 Scam Education Library
- 🎮 Interactive Safety Quiz with XP System
- 📢 Live Community Scam Alerts (Firebase)
- 📊 State Scam Threat Map
- 🌐 Multilingual: English, Hindi, Gujarati
- 🌙 Dark Mode + Mobile-First Responsive
- ⚡ Progressive Web App (PWA)

## 🛠️ Tech Stack
- **Frontend**: React 18 + Vite + Tailwind CSS
- **Backend**: Node.js + Express
- **Cloud**: Firebase Firestore + Anonymous Auth
- **AI**: Rule-Based Detection Engine + Tesseract.js OCR

## 🏃 Run Locally

### Backend
```bash
cd backend
npm install
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Open **http://localhost:5173**
